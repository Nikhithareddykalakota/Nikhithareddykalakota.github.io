UX/UI Design2022 · B2B SaaS · Web Dashboard

SaaS Dashboard
Optimization

A UI/UX optimization of a B2B SaaS admin dashboard, reducing cognitive load, enhancing data visualization, and improving operational efficiency for enterprise users.

Role
UX/UI Designer
Platform
B2B SaaS Web Dashboard
Focus
Cognitive Load · Data Viz
Tools
Figma · Miro

The Problem

Enterprise users drowning in data

B2B admin users managing complex datasets faced a cluttered dashboard showing everything at once, leading to high cognitive load, missed priorities, and slow task completion for even routine actions. The interface was built for data completeness, not user comprehension.

Before & After

What changed, and why

Before
20+ widgets visible simultaneously. All data shown at same visual weight. Critical metrics impossible to distinguish from routine information.
After
Modular layout where users pin their top 5 metrics. Clear visual hierarchy: critical KPIs large, secondary data smaller, details on demand.
Before
Charts with 15+ categories, no legend, missing context for every data point. Red/green anxiety coding.
After
Focused charts with 5, 7 categories. Consistent color system with legend. Contextual tooltips explaining each data point.

The Process

From audit to optimized system

Audit
5 core pain points through task analysis
Analyzed common admin workflows, mapped each task to cognitive load imposed. Identified that 80% of user value came from just 5 of the 20+ available actions.
Define
Cognitive load mapping and workflow analysis
Prioritized the 10 most common admin tasks and designed the hierarchy around them. Defined a power user mode and an overview mode for different user types.
Design
Modular dashboard + custom reporting
Modular card system where admins drag and pin most-used metrics. Custom reporting tools let users build exact charts they need instead of exporting raw data.
Deliver
Streamlined navigation for power users
Keyboard shortcuts for the 5 most common actions. Sidebar reorganized by task frequency rather than feature category. Quick-action panel for routine operations.
