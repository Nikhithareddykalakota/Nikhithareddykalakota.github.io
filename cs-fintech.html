UX/UI Design2022 · Personal Finance · Mobile App

Fintech User
Experience

A secure, trust-first interface for a personal finance management app, combining transparency, clarity, and intuitive budgeting tools for users handling sensitive financial data.

Role
UX/UI Designer
Platform
Personal Finance Mobile App
Focus
Trust Design · Data Visualization
Tools
Figma · Miro

The Problem

Users felt anxious, not empowered

Users handling sensitive financial data felt overwhelmed by cluttered interfaces lacking clear hierarchy and trust signals. The existing experience prioritized data density over user confidence. The goal: make every interaction feel secure, understandable, and in control.

Key Features

Built around trust and clarity

Budgeting Tools
Visual spending categories with clear progress indicators and monthly comparison
Investment Tracking
Portfolio overview with simplified performance charts and plain-language context
Custom Reporting
User-configurable spending reports with export options for tax season
Secure Data Display
Sensitive numbers masked by default with one-tap reveal and biometric auth
Modular Layout
Dashboard widgets users can rearrange based on what matters most to them
Simplified Onboarding
3-step setup with progressive disclosure, no overwhelming forms on day one

The Process

Trust-first design thinking

Research
Trust gap analysis and competitor audit
Analyzed leading fintech apps and identified common failures: jargon-heavy language, unclear data ownership, and anxiety-inducing color coding without context.
Define
4-pillar trust framework
Transparency (show data usage) · Control (hide sensitive numbers) · Clarity (plain language) · Consistency (same patterns across all screens).
Design
Modular dashboard and clear data hierarchy
Modular dashboard where users pin their top metrics. Data visualization avoids red/green anxiety coding. Numbers masked by default with clear reveal interaction.
Iterate
Onboarding reduced from 12 screens to 3
Progressive disclosure so users see only what they need at each moment. Financial literacy tooltips appear contextually rather than in a separate help section.
